## Installation
* Copy the `vimrc` file to `~/.vimrc`
* Install Vundle with `git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim` from https://github.com/VundleVim/Vundle.vim
* Run `vim +PluginInstall +qall`

Enjoy ;)
